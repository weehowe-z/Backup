1. 使用说明

直接运行

./tank_game/GLTank.exe

操作 : 

w - 向前移动
a - 向左旋转
s - 向后移动
d - 向右旋转
鼠标移动 - 改变炮口位置 
鼠标左键单击 - 射击

游戏方式 :

击中敌方坦克三次即可消灭该单位，控制台会统计战况。

编译并运行

开发环境：

  • OS - Windows 10 Home
  • IDE - Visual Studio 15 Community
    低版本的VS可能需要重新创建工程

2. 其他说明

  • glm.c/glm.h 来自开源代码，用于读取 obj文件

  • tga.c/tga.h 来自开源代码，用于读取 tga文件

  • 模型使用 Windows 10 自带工具 3D Builder 处理

3. 特点

  • 尝试采用模块化设计，降低耦合度
  • 光照，贴图，坐标变换等等均有涉及

4. 后续计划

  • 添加碰撞检测
